## 2. Course purpose and storyline

This course is a structured apprenticeship into psychological literacy. Students learn to explain behavior and mental processes using core psychological models and to evaluate claims using methods, measurement, and warranted inference. Students practice distinguishing strong evidence from weak evidence, and they learn to apply ethical and contextual constraints when interpreting findings and making decisions.

By the end of the course, students can take a claim about behavior, mind, or health and do the following consistently:
- define the relevant construct in concrete terms
- select or critique an appropriate method or design
- interpret evidence with measurement and statistical constraints
- identify alternative explanations and boundary conditions
- apply ethics and context as constraints on conclusions
- state conclusions with calibrated confidence and clear limits

### Exit tickets
- Format: 2 multiple choice items plus 1 short answer
- Short answer: 2–4 sentences, worth 2 points
- Applicability: class meetings that are not exam days and not no-class days
- Evidence use: same-day instructional adjustment using deterministic triggers

### Quizzes
- Format: 15 multiple choice items
- Composition: 9 mastery items and 6 spiral items
- Spiral composition: 2 one-interval, 2 three-interval, 1 six-interval, 1 foundations
- Release: after Thursday class
- Due: Sunday 11:59 p.m. Central
- Suppression: no quiz is released when the exam is on Thursday

## 7. Student-facing operating policies

This section is intentionally minimal. It is designed to be enforceable without creating extra operational burden.

### Attendance
- Attendance is recorded each class meeting.
- Attendance points are determined by the attendance category rules used for the term.

### Late work and make-ups
- Quizzes: due Sunday 11:59 p.m. Central.
- Missed quiz: handled according to the quiz and make-up policy you choose for the term.
- Exams: make-up eligibility and format must be specified and applied consistently.

### Academic integrity
- Students complete quizzes and exams using the allowed resources for that assessment.
- Sharing or posting quiz or exam items or answers is not permitted.

### AI use
- AI tools may be used for studying and practice unless you explicitly prohibit them.
- AI tools may not be used to produce graded written work unless you explicitly permit it.
- If AI use is allowed for a specific task, the allowed use and the expected student output are stated in the task directions.

### Accommodations and accessibility
- Accommodations may change time, location, or response format.
- Accommodations do not change what the assessment is intended to measure.
- Tasks do not require personal disclosure to succeed.

## 8. Weekly instructional routine

### Tuesday default pattern
- Build the model and vocabulary for the topic.
- Require one constructive student output.
- Run at least one check for understanding with a decision rule.
- Exit ticket.

### Thursday default pattern
- Extend and stress-test the model using examples and distinctions.
- Prefer interactive practice with a visible output and a debrief.
- Exit ticket.
- Quiz release after class when the week is not an exam week

## 9. Artifact production and acceptance criteria

For each instructional meeting that is not an exam day and not a no-class day, the minimum meeting-ready artifact set is:
- Slides
- Speaker notes or speaker script
- Guided notes
- In-class activity prompt
- Exit ticket

Acceptance criteria are enforced by the interoperability invariants:
- shared vocabulary subset
- shared mechanism map
- shared throughline and takeaway sentence
- aligned misconceptions and fast tests

## 10. Evidence review and deterministic adjustment

### Exit ticket thresholds
- If an exit ticket multiple choice item is missed by 35 percent or more of students, the next class includes a 2–4 minute micro-reteach and one new worked example.
- If both multiple choice items are missed by 35 percent or more of students, the next class begins by reframing the throughline and rerunning the mechanism step-by-step on a fresh example.
- Short answer is worth 2 points:
  - one point for accurate concept use
  - one point for a specific connection plus explanation
  If 25 percent or more of students lose the connection plus explanation point, the next class includes brief Claim–Evidence–Reasoning modeling and a short revision opportunity.

### Evidence capture rule
- One adjustment note is recorded per class meeting where an exit ticket is collected.
- One post-quiz note is recorded per quiz release.
