# Unit 3: Learning and Memory

## Unit study brief

### Big ideas
- Classical conditioning links stimuli; operant conditioning links behavior to consequences.
- Reinforcement increases behavior; punishment decreases behavior but does not teach alternatives by default.
- Schedules of reinforcement shape persistence.
- Observational learning shows learning without direct reinforcement.
- Memory strengthens with spaced retrieval, not passive review.

### Common traps
- Confusing negative reinforcement with punishment.
- Assuming punishment teaches the desired behavior.
- Treating reinforcement as the same as ‘reward.’
- Treating memory like a perfect recording.
- Overgeneralizing one learning example to all learning.

### Key terms
- **Classical conditioning**: Learning by associating stimuli so one predicts another.
- **Operant conditioning**: Learning by consequences of behavior.
- **Reinforcement**: Consequence that increases behavior.
- **Punishment**: Consequence that decreases behavior.
- **Negative reinforcement**: Increasing behavior by removing an unpleasant stimulus.
- **Shaping**: Reinforcing steps toward a target behavior.
- **Extinction**: Learned response decreases when reinforcement/pairing stops.
- **Observational learning**: Learning by watching others.
- **Retrieval practice**: Strengthening memory by recalling.
- **Spacing effect**: Practice spread out over time improves retention.
