### Common misconceptions

- Memory is a literal recording of events.
- Forgetting means information was never stored.
- If you cannot recall something, it was not encoded at all.
- Working memory and short-term memory are identical and can be summarized by one fixed number.
- Long-term memory is unlimited and perfectly accurate if you try hard enough.
- Confidence guarantees accuracy in eyewitness memory.
- Vividness and detail guarantee accuracy.
- Misinformation effects mean all memory is useless or all testimony is unreliable.
- Interference is the same as decay and time alone is the main cause of forgetting.
- Priming and procedural learning require conscious recall to show learning.
- Reconsolidation means memories can be easily erased on demand.
