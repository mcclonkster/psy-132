### Common misconceptions

- Learning is the same as maturation or instinct.
- Classical conditioning is the same as operant conditioning.
- Reinforcement always means a reward.
- Negative reinforcement means punishment.
- Punishment is the best way to create lasting behavior change.
- Extinction erases learning permanently.
- If a behavior returns after extinction, extinction failed.
- Observational learning requires direct reinforcement of the observer.
- Modeling is simple mimicry without attention or memory.
- Variable reinforcement means behavior is random rather than persistent.
- Latent learning means learning is unconscious magic rather than learning revealed later.
