# Unit 4: Cognition and Language

<!-- GENERATED:UNIT-BRIEF-START -->
## Unit study brief

### Big ideas
- Cognition includes attention, working memory, problem solving, and decision making.
- Heuristics speed judgments but produce predictable biases.
- Intelligence is measured and interpreted with limits; scores are not worth.
- Language has structure and develops via biology + environment.
- Cognitive load matters: working memory is limited.

### Common traps
- Assuming intelligence is fixed and singular.
- Treating tests as measures of value.
- Confusing working memory with long-term memory.
- Treating heuristics as always irrational rather than trade-offs.
- Assuming language differences equal deficits.

### Key terms
- **Working memory**: Limited-capacity system for holding/manipulating info briefly.
- **Heuristic**: Mental shortcut that speeds decisions but can increase errors.
- **Confirmation bias**: Favoring information that supports existing beliefs.
- **Algorithm**: Step-by-step procedure that guarantees a solution if followed.
- **Validity**: Whether a measure captures what it claims.
- **Reliability**: Consistency of a measure.
- **Phoneme**: Basic sound unit in a language.
- **Syntax**: Rules for sentence structure.
- **Semantics**: Meaning of language.
- **Cognitive load**: Total mental effort used in working memory.

### Key examples
- **Availability heuristic**: vivid examples distort risk judgments | Student move: Predict when judgments will be biased.
- **Ambiguous sentence**: context shapes interpretation | Student move: Explain two meanings and the context cue.

### Key studies
- **Tversky & Kahneman biases work**: Shortcuts produce systematic judgment errors. | Why it matters: Anchors bias claims in evidence.

### Activities
- **Bias spotter**: Identify a bias in scenarios and propose one correction. | Evidence: Discrimination + application.
